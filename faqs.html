Que es Auxtral?

Auxtral es un sistema operativo libre y gratuito basado en Debian GNU/Linux diseñado para su uso en PCs de escritorio y portátiles.

 

Auxtral es adecuado para particulares?

Absolutamente. Auxtral se diseña para ser cómodo y fácil de usar, pero también potente y configurable. Todo está hecho para mejorar la experiencia del usuario.

.

Auxtral es adecuado para empresas?

Por supuesto que si. Auxtral proporciona versiones de soporte a largo plazo (LTS) que son compatibles con una duración de 5 años.

 

Cuales son los requirimientos de sistema para ejecutar Auxtral?

- Procesador INTEL o similar 64 bits 2.5 GHz (i3 dual 2.9 o 3.0 GHz mínimo recomendado)

- 4 GB de RAM (se recomiendan 8 GB para un uso cómodo).

- 20 GB de espacio en disco (se recomiendan 70 GB)

- Resolución 1360 x 768 (se recomienda 1920 x 1080)

 

Puedo usar Snap y Flatpack en Auxtral?

Sí, podés usarlos. Pero Auxtral no los trae soportado por defecto, (y de hecho nunca recomendaría este tipo de paquetería) para ello tendrías que hacer lo siguiente en la terminal como superusuario:

# apt install flatpak gnome-software-plugin-flatpak

# flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

Reiniciar el sistema.

Para Snap:

# apt install snapd

# snap install core

en este link https://snapcraft.io/store podes visitar el contenido de Snap Repo

 

Auxtral tiene soporte para UEFI?

- Sí, a partir de Debian (Bullseye) 11.2 y Auxtral (Alsephina) 3.2 soprta UEFI

 

Si elijo una versión con determinado escritorio (DE), puedo instalar aplicaciones de otro entorno?

- Sí, podés, es mas, eso forma parte un poco de la filosofía de Auxtral en cuanto a las aplicaciones preinstaladas, por ejemplo, el "editor de texto" Pluma, es propio del entorno "Mate", pero viene como predeterminado en todas las versiones de Auxtral. Esta distro, no trae preinstalado ningún "visor de documentos", por la simple razón de que no sè si lo nesecitás!!!; cuando instales uno, y si elegiste la versión con DE Mate, eso no implica de ninguna manera que tengas que usar "Atril", perfectamente podés instalar "Okular" (de KDE) si este te satisfase más. Nunca voy a entender a los "puristas" de un sistema como GNU/Linux.

 

Puedo usar Auxtral con propósitos comerciales?

- Sí, absolutamente.

 

Puedo vender Auxtral?

- Sí, podés vender Auxtral sin permiso. Lo que no tenés que hacer es pretender mostrarte como parte de Auxtral o dejar que la gente entienda que estás asociado a Auxtral.

 

Puedo instalar Auxtral en las computadoras que vendo a mis clientes?

- Si. pero Auxtral (al igual que Debian) no tiene instalador en modo OEM, lo que no implica que puedas hacerlo.


Puedo usar el logotipo de Auxtral y / o capturas de pantalla?

- Si. Podés usarlos, promover y mostrar Auxtral, capturas de pantalla de Auxtral o su logotipo en artículos, revistas, sitios web, libros, diseños, películas o cualquier documento siempre que no te muestres como parte de Auxtral y que no permitas que la gente suponga que estás asociado a Auxtral.

Ver aquí >> http: //www.auxtral.com.ar/licencias.htm bajo el título: "A) De este sitio web".

 

Auxtral incluye controladores propietarios?

- No. Los controladores propietarios, como los ATI o Nvidia, son fáciles de instalar, pero no se instalan de forma predeterminada en Auxtral como tampoco ningún paquete que no sea software libre.

 

Y que pasa, o que posturs tiene Auxtral respecto del software sujeto a propiedad o de código cerrado?

- Auxtral no apoya ninguna postura política o ideológica contra ningún programa de software, contra su creador o editor, independientemente de la licencia que estos utilicen.

Entendiendo que todo el software incluído en las ISO de Auxtral es "software libre", gratuito y de código abierto, y que creo en este software como una maravillosa herramienta y una opción inteligente al alcance de todos, de ninguna manera considero que el "software libre" deba ser una limitación.

Es extraordinario compartir código, dar crédito a la tecnología que el software libre pone a nuestro alcance, profesar que las personas se ayuden, permitir que otros construyan sobre lo que hacemos y facilitar el progreso en general. Pero siempre respetando a la libertad de elegir de todas y cada una de las personas.
