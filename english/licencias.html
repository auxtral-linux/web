B) Of the software distribution and its ISO image

I. About packages from Debian

a) All the software packages included in the distribution called "Auxtral 4 (polaris Australis)", whose ISO images bear the names of:

"auxtral_4-xxxxx-F-xxx-xxxxxx.iso", except those specified below under the heading "II. Packages from Other Sources", come from the "main" section of the "official Debian 1 (Bookworm) repositories )", and are therefore distributed under the same license terms that Debian does for each of them.

b) Their list is in the "package.txt" file included in the root of the ISO image.

c) The "source code" of the packages is obtained by executing the command in a terminal:

$ apt-get source package-name

For example, to obtain the source code of "VLC" we execute:

$ apt-get source vlc
